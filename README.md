# ❓What's MC++?

MC++ is a high grade language developed for making Minecraft commands easily. Nowadays, MC++ is supporting functions and variables definement and callment.

# ⚙ Syntax
You don't have to type it's because MC++ is supporting only int type.
## 🧩 Function definement
```
fn foo(bar, buz) {
  ...
}
```
## 📦 Variable binding
```
foo = ...;
```
